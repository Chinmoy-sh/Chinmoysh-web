# Chinmoy Kumar Biswas - Personal Portfolio

Welcome to the official repository for the personal portfolio of **Chinmoy Kumar Biswas**, CEO and Software Engineer.

Live Website: [https://chinmoysh.me/](https://chinmoysh.me/)

Designed with relentless simplicity and deep intention, this minimalist, terminal-inspired landing page strips away the noise to focus on clarity and engineering precision.

## 🚀 Features

* **Interactive Particle Network:** A highly optimized, custom-built particle animation background using the native HTML5 `<canvas>` API. Includes interactive mouse repulsion and dynamic node linking.
* **Real-Time Precision Clock:** A dynamic UTC clock pinned to the bottom of the viewport.
* **SEO & Social Ready:** Fully equipped with Open Graph (OG) tags, Twitter Cards, and JSON-LD Person Schema for optimal search engine visibility and polished social media previews.
* **Responsive & Modular:** Structured with separation of concerns in mind. Scalable file architecture utilizing **Tailwind CSS**.
* **Zero-Build Architecture:** No complex build steps or dependencies. It runs perfectly in any modern browser right out of the box.

## 🛠 Tech Stack

* **HTML5** - Semantic markup and accessibility.
* **CSS3 / Tailwind CSS** - Utility-first styling via CDN for rapid, consistent UI rendering.
* **Vanilla JavaScript** - Lightweight, native logic without the overhead of heavy frameworks.

## 🏁 Getting Started

Since this project relies on native web technologies without a build pipeline, running it locally is incredibly simple.

1. **Clone the repository:**

    ```bash
    git clone https://github.com/Chinmoy-sh/Chinmoysh-web.git
    cd Chinmoysh-web
    ```

2. **Launch the site:**
    Simply open the `index.html` file in any modern web browser.
    *Alternatively, you can use a local development server like VS Code's "Live Server" extension for hot reloading.*
